# coupang-seattle-dec-2025
A repository supporting on-site training in Seattle, 2025


# Prerequisites: Download Necessary Software
To complete this course, you'll need the following installed on your device:

The Temporal executable. Follow the Set up a local development environment for Temporal instructions to install Temporal on your device
Git. If you don't have Git installed, follow the Installation instructions in the Git documentation.
The Java programming language installed. This course was tested with Java 20, however, any version of Java that is supported by the Temporal SDK will suffice.
The Maven project management software installed. You can use Gradle or another software of your choice, but this course's examples will use Maven.
A text editor. Visual Studio Code is a good choice, but any text editor that supports Java will work.
